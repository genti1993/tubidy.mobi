# Tubidy - Klon për Shkarkime MP3 & MP4
## 🚀 Veçoritë
- **Dark Mode Automatike**: Ndryshon temën vetvetiu sipas sistemit të përdoruesit.
- **I optimizuar për SEO**: Përmban të gjitha kodet e nevojshme për renditje në Google dhe shpërndarje në rrjete sociale.
- **I shpejtë**: Nuk përdor librari të rënda, kodi është i pastër.

## 🛠️ Si ta publikoni në internet falas
1. Ngarkoni (Push) këto skedarë në GitHub.
2. Shkoni te **Settings** > **Pages** në faqen e projektit tuaj.
3. Te sekshoni **Build and deployment**, zgjidhni `Deploy from a branch`.
4. Zgjidhni degën `main` dhe klikoni **Save**.
